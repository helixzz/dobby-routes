# Dobby Routes

Generate optimized China mainland IP route tables from authoritative sources (APNIC, GitHub).

## Installation

```bash
git clone https://github.com/your-username/dobby-routes.git
cd dobby-routes
pip install -e .
```

Or install dependencies directly:

```bash
pip install -r requirements.txt
```

## Usage

```bash
# Generate all route tables (default output to ./output/)
dobby-routes

# Specify output directory
dobby-routes --output-dir /path/to/output

# Skip GitHub sources (APNIC only)
dobby-routes --skip-github

# Skip APNIC source (GitHub only)
dobby-routes --skip-apnic

# Verbose logging
dobby-routes -v
```

## Output Files

The tool generates three files:

### `cn_routes_annotated.txt`

Forward route table with operator annotations (ASN and name).

```
# China mainland IP route table (annotated)
# Generated by dobby-routes on 2025-04-13 12:00:00 UTC
# Total routes: 8432
#
1.0.1.0/24  # AS4134/AS4812 China Telecom
1.0.2.0/23  # AS4134/AS4812 China Telecom
1.1.0.0/24  # CN
1.1.4.0/22  # AS4837/AS9929 China Unicom
```

### `cn_routes.txt`

Forward route table, fully optimized (merged and deduplicated). No comments.

```
# China mainland IP route table (optimized)
# Generated by dobby-routes on 2025-04-13 12:00:00 UTC
# Total routes: 6120
#
1.0.1.0/24
1.0.2.0/23
1.0.8.0/21
```

### `cn_routes_inverse.txt`

Inverse route table — the complement of the forward routes within the entire IPv4 address space (`0.0.0.0/0`). Fully optimized.

```
# Inverse route table (complement of China mainland routes)
# Generated by dobby-routes on 2025-04-13 12:00:00 UTC
# Total routes: 5890
#
0.0.0.0/5
8.0.0.0/7
```

## Data Sources

| Source | URL | Update Frequency |
|--------|-----|------------------|
| APNIC Delegated | `ftp.apnic.net/stats/apnic/delegated-apnic-latest` | Daily |
| China Telecom | `gaoyifan/china-operator-ip` (chinanet.txt) | Daily |
| China Unicom | `gaoyifan/china-operator-ip` (unicom.txt) | Daily |
| China Mobile | `gaoyifan/china-operator-ip` (cmcc.txt) | Daily |
| CERNET | `gaoyifan/china-operator-ip` (cernet.txt) | Daily |
| chnroutes2 | `misakaio/chnroutes2` | Hourly |
